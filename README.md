# tools

Personal Windows utilities and scripts. Each tool lives in its own folder
with source and build instructions. Compiled downloads are in
[Releases](../../releases). Listed at dayhaw.dev/tools.

## Tools

| Tool | Description | Download |
|------|-------------|----------|
| [audioswap](audioswap/) | Single-button toggle of the Windows default playback device between two configured endpoints | [audioswap v1.0](../../releases/tag/audioswap-v1.0) |
| [setres](setres/) | Per-monitor resolution switcher, immune to display index shuffling | [Latest](../../releases/tag/setres-v1.0) |
| [tvmode](tvmode/) | One-command desk/couch mode switcher for display, audio, and Samsung TV input | [Latest](../../releases) |
